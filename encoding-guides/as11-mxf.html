Vue d'ensemble

Afin de créer une sortie AS-11, définissez le format_preset paramètre à "as11_hd". Actuellement, l'audio est limité à l'audio PCM à 2 canaux, avec une fréquence d'échantillonnage maximale de 48 kHz. Pour plus d'informations sur les sorties AVC-Intra, consultez la documentation ici.

  • format_preset: Réglez ce paramètre sur as11_hd pour la sortie ae AVCi 100 MXF.
  • audio_sample_rate: La valeur par défaut n'est pas définie. Limité à 48000.

Exemple de paramètres

Exemple de requête AS-11

{
    "input": "s3://zencodertesting/test.mov",
    "output": {
        "format_preset": "as11_hd",
        "audio_sample_rate": 48000
    }
}

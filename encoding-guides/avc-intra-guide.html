Présentation d'AVC-Intra

AVC-Intra définit deux préréglages, AVC-Intra 50 et AVC-Intra 100. AVC-Intra 50 est codé à un débit de 50 mbit / s tandis que AVC-Intra 100 est codé à 100 mbit / s. Utilisez le video_codec_preset paramètre pour définir une sortie AVCi. Pour plus d'informations sur la création de sorties AS-11, consultez la documentation ici.

  • video_codec_preset: Règle le préréglage AVC-Intra pour les sorties. Valeurs : avci_50, avci_100.

Exemple de paramètres

Exemple de requête AVC-Intra 50

{
    "input": "s3://zencodertesting/test.mov",
    "output": {
        "video_codec_preset": "avci_50"
    }
}

Exemple de requête AVC-Intra 100

{
    "input": "s3://zencodertesting/test.mov",
    "output": {
        "video_codec_preset": "avci_100"
    }
}

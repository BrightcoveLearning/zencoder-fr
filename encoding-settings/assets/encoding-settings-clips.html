start_clip

Start_Clip : String

Versions de l'API : V1, V2

Parent : sorties

Valeurs valides : Un code temporel (HH:MM:SS.S), ou un nombre positif représentant un nombre de secondes

Types de tâches compatibles : vod

Exemple :

  • 00:00:10 .0
  • 01:31:08 .3
  • 60
  • 12,5

Description :

Créez un sous-clip de la vidéo originale, en commençant à un moment donné. Il peut s'agir d'un code temporel au format HH:MM:SS.S ou d'une décimale exprimant le nombre de secondes à partir de laquelle commencer. Notez que l'heure de début réelle peut ne pas être parfaitement exacte, selon la disposition des images clés dans la vidéo.

{ "input": "s3://zencodertesting/test.mov", "outputs": [ { "start_clip": "00:00:10.0" }] }

clip_length

Clip_Length:String

Versions de l'API : V1, V2

Parent : sorties

Valeurs valides : Une durée au format HH:MM:SS.S ou un nombre positif représentant un nombre de secondes

Types de tâches compatibles : Tous

Exemple :

  • 00:00:10 .0
  • 01:31:08 .3
  • 60
  • 12,5

Description :

Créez un sous-clip d'une longueur particulière. Il peut s'agir d'une durée avec un format HH:MM:SS.S ou d'une décimale exprimant la longueur du clip en secondes. Peut être combiné avec start_clip pour créer des sous-clips au milieu d'une vidéo. Si start_clip n'est pas fourni, le clip démarrera au début de la vidéo. Notez que la longueur réelle du clip peut ne pas être parfaitement exacte, selon la disposition des images clés dans la vidéo. Notez également que clip_length est une durée et non le code temporel du fichier multimédia auquel le sous-clip doit se terminer.

{ "input": "s3://zencodertesting/test.mov", "outputs": [ { "clip_length": "00:00:10.0" }] }

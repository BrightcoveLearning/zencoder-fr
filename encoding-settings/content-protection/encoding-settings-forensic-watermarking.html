Introduction

Le filigrane médico-légal aidera à protéger votre contenu premium contre le piratage et le partage de contenu non autorisé. Cette fonctionnalité permet également d'identifier rapidement la source d'une fuite de contenu, afin que des mesures puissent être prises. Voir Filigrane médico-légal pour plus de détails sur les exigences relatives à l'utilisation du filigrane médico-légal.

forensic_watermark

forensic_watermark:String

Versions d'API :V2

Parent : sorties

Valeurs valides : "UN B"

Doit être réglé sur A pour une sortie, et B pour une deuxième sortie

Types de tâches compatibles : vod

Exemple : UN

Description :

Pour générer des sorties avec filigrane judiciaire dans Zencoder, vous devez spécifier 2 sorties par vidéo, une sortie UNE et B afin que les segments de chaque sortie puissent être mélangés dans différentes séquences pour produire une gamme de différentes « vidéos » filigranées pour la lecture (par exemple : AABABBABA, ABBBABABA, BABAABBAA, etc.).

Remarques :
  • filigrane médico-légal ne peut pas être ajouté à des vidéos de moins de 5 minutes
  • segment_seconds doit être prêt à 2 ou le filigrane médico-légal ne sera pas ajouté

forensic_watermark_stub_mode

forensic_watermark_stub_mode:Boolean

Versions d'API :V2

Parent : sorties

Valeurs valides : vrai faux

Valeur par défaut : false

Types de tâches compatibles : vod

Exemple : vrai

Description :

À être utilisé à des fins de test des intégrations qui génèrent des sorties avec filigrane médico-légal dans Zencoder. Lorsqu'il est défini sur true avec forensic_watermark, cet indicateur provoque la création de filigranes visibles (à vérifier que le filigrane médico-légal a été ajouté avec succès).

Une fois les tests terminés, vous devrez exécuter à nouveau la tâche, en omettant l' forensic_watermark_stub_mode indicateur ou en le définissant sur false, pour obtenir des sorties sans le paramètre visible. filigranes.

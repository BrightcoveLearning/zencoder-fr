Recommandations d'encodage

AV1 est un format d'encodage vidéo de nouvelle génération ouvert et libre de droits de l'Alliance of Open Media Video. Il peut fournir une qualité supérieure à des débits débits nettement inférieurs à ceux des codecs HEVC, VP9 et H.264.

Voir le Guide AV1 pour plus d'informations, et portez une attention particulière à Limites pour le support actuel.

En plus des paramètres de cette page, vous pouvez inclure d'autres paramètres vidéo généraux, mais notez que les paramètres suivants ne sont pas pris en charge actuellement :

  • video_reference_frames
  • video_bframes
  • max_video_bitrate
  • quality
  • crf
  • decoder_bitrate_cap

Notez également que pour les paramètres de résolution (que vous utilisiez size ou width et height) la largeur et la hauteur doivent être multiples de 8.

video_codec_level

video_codec_level:String

Versions de l'API : V2

Parent : sorties

Valeurs valides :

  • 2.0
  • 2.1
  • 3.0
  • 3.1
  • 4.0
  • 4.1
  • 5.0
  • 5.1
  • 5.2
  • 5.3
  • 6.0
  • 6.1
  • 6.2
  • 6.3

Types de tâches compatibles : Tous

Exemple : 4.1

Description :

Contraint le débit binaire et les unités d'arborescence de codage (CTU) (AV1).

Notez que ce paramètre peut être utilisé pour d'autres codecs avec des valeurs autorisées différentes. Voir Paramètres d'encodage vidéo.

{
  "input": "s3://zencodertesting/test.mov",
  "outputs": [
    {
      "video_codec_level": "5.1"
    }
  ]
}

video_codec_profile

video_codec_profile:String

Versions de l'API : V2

Parent : sorties

Valeurs valides : main, high, professional

Par défaut : main

Types de tâches compatibles : Tous

Exemple : 4.1

Description :

Définit le profil de codage.

Notez que ce paramètre peut être utilisé pour d'autres codecs avec des valeurs autorisées différentes. Voir Paramètres d'encodage vidéo.

{
  "input": "s3://zencodertesting/test.mov",
  "outputs": [
    {
      "video_codec_profile": "high"
    }
  ]
}

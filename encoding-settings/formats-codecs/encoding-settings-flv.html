cue_points

cue_points:Array

Versions de l'API : V1, V2

Parent : sorties

Valeurs valides : Un tableau de hachages avec des paramètres de point de repère

Types de tâches compatibles : vod

Description :

Zencoder prend en charge l'ajout de points de repère d'événement et de navigation aux fichiers VP6 Flash Video (FLV). Les points de repère sont intégrés dans le flux FLV et le paquet de métadonnées FLV lorsque vous encodez le fichier FLV.

Il existe deux types de points de repère : « navigation » et « événement ». Les points de repère de navigation peuvent être utilisés pour permettre aux utilisateurs de rechercher une partie spécifiée d'un fichier vidéo. Les points de repère de navigation créent une image clé à l'emplacement du point de repère spécifié, vous pouvez donc utiliser du code pour déplacer la tête de lecture d'un lecteur vidéo vers cet emplacement. Vous pouvez définir des points particuliers dans un fichier FLV dans lesquels vous pouvez souhaiter que les utilisateurs recherchent. Par exemple, votre vidéo peut avoir plusieurs chapitres ou segments, et vous pouvez contrôler la vidéo en incorporant des points de repère de navigation dans le fichier vidéo.

Les points de repère d'événement peuvent être utilisés pour effectuer des actions dans un lecteur vidéo SWF qui sont déclenchées à des points spécifiés pendant la lecture FLV.

Pour plus d'informations, consultez ceci Article Adobe sur les points de repère.

{
  "input": "s3://zencodertesting/test.mov",
  "outputs": [
    {
      "cue_points": [
        {
          "type": "navigation",
          "time": 30.4,
          "name": "My Cue Point",
          "data": {
            "mykey": "myvalue"
          }
        }
      ]
    }
  ]
}

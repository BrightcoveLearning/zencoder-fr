indice

hint:Boolean

Versions de l'API : V1, V2

Parent : sortie

Valeur par défaut : false

Valeurs valides : vrai ou faux

Types de tâches compatibles : Tous

Exemple : vrai

Description :

Activez l'indication des fichiers MP4 pour RTP / RTSP. La taille mtu_size par défaut est de 1450 octets. Utilisez l'option mtu_size pour modifier cette option.

{
  "input": "s3://zencodertesting/test.mov",
  "outputs": [
    {
      "hint": true
    }
  ]
}

mtu_size

mtu_size:Integer

Versions de l'API : V1, V2

Parent : sortie

Par défaut : 1 450

Valeurs valides : Un entier entre 100 et 5000

Types de tâches compatibles : Tous

Exemple : 2 000

Description :

Taille du MTU en octets pour les indices MP4.

{
  "input": "s3://zencodertesting/test.mov",
  "outputs": [
    {
      "mtu_size": 2000
    }
  ]
}

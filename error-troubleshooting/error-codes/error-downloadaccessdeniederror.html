Signification

Nous avons essayé de télécharger un fichier à l'adresse spécifiée, mais nous avons obtenu une erreur d'autorisation. S'il s'agit d'une URL S3, vous devez probablement accorder un accès en lecture à Zencoder pour ce compartiment ou ce fichier. La façon la plus simple de le faire est d'utiliser un client S3, comme AWS Console d'Amazon.

En termes techniques, cela signifie que nous avons obtenu une réponse 403 Accès refusé du serveur.

Solutions de contournement

Assurez-vous que les autorisations sur le fichier à l'URL donnée autorisent l'accès Zencoder. Si le fichier est sur S3, vous devrez peut-être accorder un accès en lecture à Zencoder.

Signification

Le débit vidéo fourni est trop faible pour la vidéo. Augmenter le débit devrait corriger le travail. Cela ne devrait être rencontré que lors de l'exécution dans strict mode, Zencoder corrigera automatiquement le problème dans le cas contraire.

Réessayer ce boulot ?

Emplois avec cette erreur ne devrait pas être retenté. En savoir plus sur la relance des tâches.

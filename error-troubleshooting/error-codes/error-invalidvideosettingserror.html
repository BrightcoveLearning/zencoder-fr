Signification

Les paramètres de codage vidéo fournis n'étaient pas valides. Certains paramètres sont corrigés automatiquement sauf si le mode strict est activé. Consultez la page d'informations de sortie pour plus de détails sur les paramètres non valides.

Réessayer ce boulot ?

Emplois avec cette erreur ne devrait pas être retenté. En savoir plus sur la relance des tâches.

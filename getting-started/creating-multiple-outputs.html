Création de plusieurs sorties

Cette rubrique explique comment créer plusieurs sorties pour une tâche d'encodage.

Créer plusieurs sorties

Vous pouvez spécifier plusieurs sorties en transmettant plus d'un hachage dans la baie de sorties, comme ceci :

{
  "input": "https://user:pass@bucket-name/file-name.avi",
  "outputs": [
    {
      "label": "web",
      "url": "s3://output-bucket/web.mp4",
      "size": "512x384"
    },
    {
      "label": "iphone",
      "url": "s3://output-bucket/iphone.m4v",
      "size": "480x320"
    }
  ]
}

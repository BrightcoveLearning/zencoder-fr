Obtenir une clé API

Toutes les requêtes à l'API Zencoder doivent être authentifiées à l'aide d'une clé API. Cette rubrique explique les différents types de clés API pour Zencoder et comment les obtenir.

Obtention d'une clé API

Pour demander une clé API, accédez à le site Zencoder. Si vous êtes déjà inscrit en tant qu'utilisateur, connectez-vous. Si ce n'est pas le cas, inscrivez-vous en tant que nouvel utilisateur. Vous ne pouvez pas demander de clés API avant d'être enregistré en tant qu'utilisateur.

Une fois connecté, accédez à la section API :

Dans la section API, vous verrez des options pour demander différents types de clés d'API :

Consultez les sections ci-dessous pour une explication des différents types de clés API. Notez que vous devez être inscrit à un forfait payant pour obtenir les clés d'accès complet.

Touches d'accès complet

Les clés d'accès complet permettent de faire n'importe quel type de requête à l'API Zencoder. C'est le genre de clé dont vous aurez besoin pour soumettre des jobs à Zencoder et obtenir des sorties.

Touches en lecture seule

Les clés d'accès en lecture seule vous permettent de créer GET demandes uniquement à l'API Zencoder. Ceux-ci sont utiles pour les utilisateurs qui ont seulement besoin d'obtenir le statut des emplois, mais qui ne soumettent pas réellement des emplois. Voir Utilisation de clés API en lecture seule pour plus d'informations.

Touches du mode d'intégration

Les touches de mode d'intégration vous permettent de soumettre des travaux, mais les tâches ne seront pas traitées. Ils sont principalement utiles aux intégrateurs système qui veulent tester les demandes d'erreurs. Ils peuvent également être utilisés pour avoir une idée de l'utilisation de l'API avant d'acheter un plan payant Zencoder, ou pour vérifier les demandes avant d'obtenir réellement des sorties.

Paramètres de sortie

Cette rubrique explique les sorties par défaut pour les travaux Zencoder et les bases de la définition de vos propres sorties.

Définition des sorties

Notre sortie par défaut est un fichier vidéo unique avec les paramètres suivants :

Vidéo H.264, audio AAC, format de fichier MP4
Vidéo : de qualité moyenne
Audio : stéréo, qualité moyenne (environ 112 kbps), 44100 Hz

Si vous ne spécifiez pas d'emplacement de sortie, le fichier sera temporairement hébergé par Zencoder. Après 24 heures, le fichier ne sera plus disponible.

Bien sûr, vous pouvez contrôler le paramètres de sortie. Pour ce faire, transmettez un tableau d'options dans la demande d'API, comme ceci :

{
  "api_key": "93h630j1dsyshjef620qlkavnmzui3",
  "input": "s3://bucket-name/file-name.avi",
  "outputs": [
    {
      "url": "s3://output-bucket/output-file-name.mp4",
      "width": "1280",
      "height": "720"
    }
  ]
}

Cet exemple montre comment créer un fichier de sortie unique à partir de la vidéo d'entrée d'origine et le charger dans un compartiment S3. Vous pouvez créer plusieurs fichiers de sortie à partir de l'original sans avoir à créer de travaux d'encodage supplémentaires.

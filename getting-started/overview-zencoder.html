Vue d'ensemble : Zencoder

Zencoder est une plateforme puissante pour transcoder la vidéo à la demande.

Introduction

Zencoder offre un service de transcodage rapide et fiable pour les fichiers vidéo et audio. Il peut produire des sorties pour le web et une grande variété d'appareils, et peut facilement être intégré à votre CMS et à d'autres applications.

L'interface utilisateur

Zencoder est principalement une API, mais il y a une interface utilisateur qui inclut :

  • Paramètres du compte
  • Tableau de bord affichant les résultats des travaux récents
  • Un générateur de requêtes API qui vous permet de commencer avec l'API sans écrire de code

L'API

Il existe plusieurs fonctions que vous pouvez effectuer à l'aide de l'API, dont les plus importantes sont la soumission et la gestion des tâches d'encodage pour la vidéo à la demande.

Pour obtenir un guide complet des points de terminaison d'API, consultez le Référence API.

La puissance de Zencoder réside dans la flexibilité qu'il vous offre pour créer exactement le type de sortie que vous voulez. C'est aussi la source d'une certaine complexité, car il existe de nombreux paramètres d'encodage à comprendre. Un bon point de départ pour les explorer est le résumé des paramètres d' encodage, qui contient des liens qui vous permettent d'accéder aux descriptions complètes avec un exemple de code.
